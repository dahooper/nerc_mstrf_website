--- layout: default title: Announcement 2006-05-10 --- Sky-camera images and laser ceilometer backscatter data now available
Since 25th April 2007, images from the new sky-camera at the MST Radar site are being captured at 1 minute intervals. Full details of this instrument, together will a sample image, can be found here.

Since 24th April 2007, the laser ceilometer has been outputting backscatter profiles in addition to the standard cloud base information. The new data will not be made available through the BADC until they have been rewritten into self-documented netCDF files (the cloud base data are already available in NASA-Ames format files). Please contact the NERC MST Radar Facility Project Scientist if you have an immediate need for the backscatter data. Full details of this instrument, together will a sample quick-look plots of both cloud base and backscatter data, can be found here.
