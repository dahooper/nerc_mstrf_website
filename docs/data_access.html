--- layout: default title: 'NERC MSTRF: access to the data' --- In keeping with NERC's open access data policy:
  • anyone may have free access to the NERC MST Radar Facility dataset
  • use of the data is covered by the (UK) Open Government Licence
  • the data may be used for any purpose, including a commercial one
  • the principal condition of use is that the source of the data MUST be appropriately acknowledged

The data will typically be made available through the Centre for Environmental Data Analysis (CEDA) within 36 hours of acquisition. This allows time for final quality control checks to be carried out. The release of data will be delayed over weekends, when the facility manager is on leave, or when there are technical problems with the data delivery.

Application for Facility Support

All users of the Facility's data are strongly encouraged to submit an Application for Facility Support (AFS). It is only by being able to report to NERC on the projects it supports that the Facility is able to ensure continued funding.

It is only obligatory to submit an AFS if you require:
  • User Support: you require any interaction with the Facility's Project Scientist e.g. for help with accessing, analysing, or interpreting the standard data products or for the provision of non-standard data products
  • Campaign Support: you require the Facility to support an observation campaign
  • Guest Instrument Support: you require the Facility to host a piece of guest equipment
Before submitting an AFS, you should contact the Facility's Project Scientist to check whether the desired support can be provided.

Data users who have not applied for Facility Support should not expect direct help with anything beyond the most trivial of enquiries. They must rely solely on the online resources. They simply need to click on the first link below in order to gain data access.
