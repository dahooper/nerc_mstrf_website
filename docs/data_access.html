In keeping with NERC's open access data policy:

  • anyone may have free access to the NERC MST Radar Facility dataset
  • use of the data is covered by the (UK) Open Government Licence
  • the data may be used for any purpose, including a commercial one
  • the principal condition of use is that the source of the data MUST be appropriately acknowledged

In order to gain access to the data:

  • Visit the NERC MSTRF dataset collection page in the Centre for Environmental Data Analysis (CEDA) catalogue
  • Select the appropriate dataset from the dropdown menu
  • Follow the Request access link if you don't already have access to the dataset and/or a CEDA account
  • Follow the Explore link if you already have access to the dataset and a CEDA account
  • Follow the More Info link if you would like more details about the dataset

Data files covering a Coordinated Universal Time (UTC) day will typically be made available during the morning following data acquisition. This allows time for final quality control checks to be carried out. The release of data will be delayed when the facility manager is unavailable (e.g. during holidays and over weekends) or when there are technical problems with the data delivery.

Application for Facility Support

An application for facility support does not (necessarily) need to be made if you simply want access to the data. However, one will be necessary if you require:

  • User Support: you require any interaction with the Facility's Project Scientist e.g. for help with accessing, analysing, or interpreting the standard data products or for the provision of non-standard data products
  • Campaign Support: you require the Facility to support an observation campaign
  • Guest Instrument Support: you require the Facility to host a piece of guest equipment

Before submitting an application, you should contact the Facility's Project Scientist to check whether the desired support is feasible. Applications must be submitted through the Atmospheric Measurement & Observation Facility website.
