EXTERNAL LINKS 		THE NERC MST RADAR FACILITY AT ABERYSTWYTH
FILE FORMAT FOR MET OFFICE GPS INTEGRATED WATER VAPOUR DATA
These files contain data products derived from measurements made by the Met Office's GPS water vapour receiver located at the NERC MST Radar site. The files are written in a non-standard ASCII format. They contain no header information - just a series of data lines which contain one hour average data products. The times given refer to the middle of that hour. Each file contains All the data prodcuts for a given month. The first 5 lines from the file for August 2001 (gps-iwv_capel-dewi_200108.txt) are shown below in green. The text in red is for explanatory purposes only and does not appear in the files. 
ABYW    2001    08      03      15      30      2429.0  137.0   21.8
ABYW    2001    08      03      16      30      2426.9  133.5   21.2
ABYW    2001    08      03      17      30      2424.6  130.7   20.7
ABYW    2001    08      03      18      30      2427.8  133.3   21.1
ABYW    2001    08      03      19      30      2435.7  141.2   22.4
SITE    YYYY    MM      DD      hh      mm        ERRC   WETD    IWV

Column Parameter Description
1 SITE Site code
2 YYYY 4-digit year
3 MM 2-digit month [01 - 12]
4 DD 2-digit day [1 - 31]
5 hh 2-digit hour [0 - 23]
6 mm 2-digit minute [0 - 59]
7 ERRC Error correction (kg/m^2)
8 WETD Wet delay (kg/m^2)
9 IWV Integrated water vapour (kg/m^2)
