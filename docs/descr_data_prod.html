DESCRIPTION OF MST RADAR DATA PRODUCTS
The following are the principal data products available from the NERC MST radar:
Horizontal wind vector (and wind shear)
Vertical velocity
Radar return signal power
Aspect sensitivity
Spectral width

Each parameter is described using observations made on 12th February 2001 by way of illustration; click here to see all data products together on one plot. The crosses, in all panels, represent the tropopause altitudes determined from the radar return signal power.
